Se generan ejemplos de Inyeccion deDependencias con las siguientes casos:

- Component-Scan
- Constructor-Arg
- Inheritance
- Setter-Inyection


Forma de ejecucion.

Cada package contiene un ejemplo de un caso, y se ejecuta la clase main que termina en **Application**:
