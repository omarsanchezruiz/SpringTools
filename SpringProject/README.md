Se generan ejemplos de Inyeccion deDependencias con las siguientes casos:
- Application-Context
- Component-Scan
- Constructor-Arg

- Setter-Inyection

Forma de ejecucion.
Cada package contiene un ejemplo de un caso, y se ejecuta la clase que termina en **Application**.
